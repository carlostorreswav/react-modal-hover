# react-modal-hover (beta)
## Quick start

### install package from npm
npm i react-modal-hover

#### import on your .js
import { ModalHover } from 'react-modal-hover'

#### Prepare your "onHover" component or just a simple string

<ModalHover onHover={<h3>Hello World</h3>}>
  <div>Hover Me!</div>
<ModalHover/>

and thats it!
